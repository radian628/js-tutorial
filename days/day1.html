Introduction

What is HTML?

HTML is a markup language (not a programming language!) that describes the content of web pages. It tells the browser that what you're reading right now is a paragraph of text, and that the text above this paragraph is a header. It can describe all sorts of elements: Paragraphs, headers, lists, images, buttons, inputs, and even videos, to name a few.

An important distinction to make with HTML is that HTML describes what is on a page, not how it looks. It will tell you that there's two paragraphs of text. However, it won't tell you how big the font is, how wide the paragraphs are, whether they're separated, et cetera. Sure, the browser creates a default look, but this should not be relied on if you're going for a specific style! Determining how a page looks is the job of CSS (which is another topic entirely).

What is JavaScript?

JavaScript is a programming language- in other words, a series of human-readable instructions that can be read and executed by a computer. JavaScript's primary purpose is adding interactivity to websites. Virtually every website you visit contains JavaScript somewhere. When you press a button, move your mouse, scroll, or even just sit idley on a page, there is JavaScript running, reacting to your every move.

Coding with JSFiddle

JSFiddle is an online code editor that lets you edit HTML, CSS, and JavaScript, all while showing you the result of what you've made. Try it out here!

There are four main tabs by default. They all do what you might expect- "HTML", "CSS", and "JavaScript" contain their respective languages, while "Result" contains the result of what you created in the other three. You can find options for running and saving fiddles near the top left corner, and you can find options near the top right. If you save a fiddle, copy the URL somewhere! That's how you can access it again!

Intro to HTML

HTML Tags

This is an HTML tag. It's the base unit of all HTML:

<tagname></tagname>

That first half, between the first set of angle brackets, is called the opening tag. The second half, between the second set of angle brackets, is called the closing tag. Anything between the opening and closing tag is said to be inside the tag. The text labeled tagname is said to be the name or type of the tag.

The HTML Template

Below is the starting template for virtually every web page in existence. It isn't strictly required, but you should still use it:

<!DOCTYPE html>
<html>
    <head>
    
    </head>
    <body>
    
    </body>
</html>

Let's break this down:

<!DOCTYPE html> - This tells the browser (and anything else reading the HTML file) that what you're looking at is, in fact, HTML. It's not exactly a "true" HTML tag.

<html> - All HTML goes in the html tag.

<head></head> - This is the head tag. You'll find "meta-information" about the web page here- for example, a description of the site that appears on search results, or a title that appears on the tab.

<body></body> - The body tag contains all the HTML on the web page that you can directly see.

Adding Content

Let's look at an example of some HTML that actually does something:

<!DOCTYPE html>
<html>
    <head>
    
    </head>
    <body>
        <p>Hello World!</p>
    </body>
</html>

The only real difference between this HTML and the template is the inclusion of the line <p>Hello World!</p>, which represents a paragraph of text, where that text is "Hello World". Try typing out this example in JSFiddle. You should get something resembling this (without the border, of course- that's just to differentiate the output from the rest of the page):

Hello World!

Here is an example with more tags:

<!DOCTYPE html>
<html>
    <head>
    
    </head>
    <body>
        <h1>I am a header!</h1>
        <p>I am a paragraph!</p>
        <button>I am a button!</button>
    </body>
</html>

Its output:

I am a header!

I am a paragraph!

As you may have guessed, the h1 tag defines a header, and the button tag defines a button.
