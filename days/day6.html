While Loops

Amongst all the talk I've been doing about repetition in code and how bad and awful that is, there's one thing that's been mysteriously missing: A way to run a segment of code a bunch of times. Sure, you can do that with a function that calls itself (similar to that fractal I showed you yesterday). However, that's weird, clunky, and it's kind of hard to tell what you're doing. Fortunately, as is often the case when something simple seems needlessly difficult in programming, a disgruntled programmer has created a solution. And the solution to running a few lines of code a bunch of times is the while loop.

What is a While Loop?

A while loop is constructed the same way you would make an if-statement. The only difference is that you replace if with while. A while loop will repeatedly run the code inside of it as long as the condition between its parentheses remains true. It only checks this condition at the end of a single cycle/iteration, so if the condition becomes false halfway through the loop, it won't suddenly stop. The way I like to think about a while loop is that it's an if-statement that's allowed to repeat indefinitely. Here's a while loop in action:


let i = 0;
while (i < 10) {
    console.log(i);
    i++;
}

This code will produce the following:

0
1
2
3
4
5
6
7
8
