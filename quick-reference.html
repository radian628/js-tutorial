Keyboard Shortcuts

Remember: replace all Ctrl with Command if on MacOS!

ShortcutUsage
Ctrl+C Copies selection to the clipboard.
Ctrl+V Pastes/inserts data from the clipboard.
Ctrl+X Deletes selection and copies it to the clipboard.
Ctrl+A Selects everything.
Ctrl+S Saves.
Arrow Key Navigate through text.
Shift+Arrow Key Same as Arrow Key, except it selects the text you navigate past.
Ctrl+Arrow Key Same as Arrow Key, except it skips whole words (PC) or whole lines (MacOS).
Ctrl+Shift+Arrow Key Combines the effects of Shift+Arrow Key and Ctrl+Arrow Key.
Tab (while text is selected) In code editors, this will indent all selected lines.
Shift+Tab (while text is selected) In code editors, this will unindent all selected lines.

Basic HTML Elements

Here are some HTML elements you should know (not comprehensive!)

Tag Name(s)Description
<!DOCTYPE html> Always start your HTML with this. It tells whatever's reading it that this is in fact an HTML document. Not really a real HTML tag.

head Contains meta-information about the document (description, title that appears on tabs, search engine info, etc.).
body Contains the actual HTML that you see as text, images, etc. on the screen.

div Generic divider (does nothing on its own; useful with JS and CSS).
ul Unordered list.
li List item. One or more of these should go directly inside a ul.

Variables

Variables can be declared with the keyword let, followed by the variable name. You can then set the variable equal to a value using = and then a value. Only use let when you use a variable for the first time!

Functions

Functions are named blocks of code that help get rid of repetition. You can create one with the keyword function, followed by the function name, parentheses (containing comma-separated arguments), and finally curly brackets containing the code. Here's a function that writes "Hello World!" to the console:

function helloWorld() {
    console.log("Hello World!");
}

Here is one that adds three numbers together, and outputs the result to the console:

function addThreeNums(a, b, c) {
    console.log(a + b + c);
}

To call a function, type out its name, followed by parentheses (regardless of whether there's arguments or not). Put the values you want the arguments to take between the parentheses. For example, addThreeNums(1, 2, 3); will print 6 to the console.

Data Types

Here are a few data types you'll frequently encounter in JavaScript:

Name Description Usage Examples
String Text; a list of characters. Put strings between single quotes (''), double quotes(""), or backticks (``).
  • 'foo'
  • "Hello World"
  • `JavaScript!`
  • "5" - This is not a number! The quotes make it a string!
Number Any number. Can be an integer, a decimal, positive or negative. Numbers can be written as-is, or in scientific notation. For scientific notation, add e+N to the end of the number, where N is the power of 10 to multiply by (e.g. e+2 means "multiply by 100"). For negative powers of 10, use e-N.
  • 5
  • -2.34567
  • 1.2e+3 - Equal to 1200
  • 35e-2 - Equal to 0.35
Boolean Represents a binary choice (two options). Can be either true or false. Booleans are written as-is, though they are often found as the result of comparison operators.
  • true
  • false
Function Represents a block of JavaScript code. Created with the keyword function, followed by parentheses containing a list of comma-separated arguments, and then curly brackets which will contain the code the function represents. 
  • function cube(x) {
    console.log(x * x * x);
}
  • function sayHello(name) {
    console.log("Hello, ");
    console.log(name);
}
